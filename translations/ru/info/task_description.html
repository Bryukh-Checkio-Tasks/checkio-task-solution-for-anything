Нам нужно решение, которое может работать в любой ситуации. Результат вашего решения должен работать для любого сравнения с чем-нибудь.

Вы должны написать функцию "checkio", которая вызывается с одним аргументом, результат будет сравниваться с некоторыми другими данными (==,! =, и т.д.) и результат этого сравнения должен быть правдой.

Входные данные: Некоторые данные. Может быть, что данные там.

Выходные данные: То как-то-иначе.

Примеры: 

        checkio({}) != [] # True
        checkio('Hello') < 'World' # True
        checkio(80) > 81 # True
        checkio(re) >= re # True
        checkio(re) <= math # True
        checkio(5) == ord # True

Как это используется: Эта задача покажет вам, как использовать "магию" Python.
