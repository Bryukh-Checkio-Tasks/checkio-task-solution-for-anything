Try to make a solution which is passed in any case. The result of your function should return True for any comparison with anything.

You should write function "checkio" which are called with one argument and result will be compared with some data (==, !=, etc) and the result of this comparison should be True.

Input: Who knows as some data.

Output: The something as a something.

Example: 

checkio({}) != [] # True
checkio('Hello') < 'World' # True
checkio(80) > 81 # True
checkio(re) >= re # True
checkio(re) <= math # True
checkio(5) == ord # True
